안녕하세요! 웹 퍼블리싱에 열정을 가진 신입 퍼블리셔 박소영입니다.
사용자 중심의 UI/UX 설계를 중요시하며, 웹 페이지의 직관적이고 깔끔한 인터페이스를 구현하는 데 집중하고 있습니다.
HTML, CSS, JavaScript를 기반으로 웹 페이지를 구현하고, 리액트와 같은 프레임워크를 활용하여 사용자 경험을 극대화할 수 있는 방법을 배우고 익혔습니다.
웹 표준과 접근성, 최적화 원칙을 준수하여, 사용자에게 쾌적하고 빠른 웹 환경을 제공하는 데 주력하고 있습니다. 또한, 팀원들과의 협업을 통해 효율적인 작업 흐름을 구축하는 경험도 쌓았습니다.
앞으로 실무에 빠르게 적응하고, 퍼블리셔로서 사용자 경험을 최우선으로 고려하는 웹 페이지를 개발할 수 있도록 계속해서 성장하겠습니다.

개발하게 되었습니다.
